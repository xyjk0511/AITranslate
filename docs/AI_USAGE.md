# AI 使用记录模板

按时间线记录与 AI 交互、生成内容以及后续修改，方便回溯。

## 记录格式
- 时间：YYYY-MM-DD HH:MM
- 操作：我向 AI 提问 / AI 给出回答 / 我做了手动修改 / 遇到问题
- 详情：描述提问、AI 返回的关键信息、手动修改内容、问题与解决方案、配图或截图说明（可贴链接或文件名）

## 示例
1. **时间**：2026-01-07 14:00  
   **操作**：我向 AI 提问  
   **详情**：请生成 FastAPI + Flutter 的翻译 Demo。
2. **时间**：2026-01-07 14:05  
   **操作**：AI 给出回答  
   **详情**：返回了后端接口和 Flutter 代码（截图）。
3. **时间**：2026-01-07 14:20  
   **操作**：我做了手动修改  
   **详情**：更新了 README 跑通步骤，修复了路径错误。
4. **时间**：2026-01-07 14:30  
   **操作**：遇到问题  
   **详情**：Android 模拟器访问 127.0.0.1 失败，改用 10.0.2.2 解决。

根据实际过程继续追加条目即可。
